# CL03.CLLE Member Guide

## Overview
ILE Control Language (CLLE) program `CL03` implements interactive or batch processes in RPGLE.

## Dependency Map
- **Incoming:** Commands or menus that invoke `CL03`.
- **Outgoing:**
  - Calls programs: WWCONDET

## Source
````cl
             PGM
             DCL        VAR(&NUM) TYPE(*DEC) LEN(6) VALUE(000001)
             DCL        VAR(&ACCT) TYPE(*CHAR) LEN(11) VALUE(' ')
             DCL        VAR(&CUSTOMER) TYPE(*CHAR) LEN(40) VALUE(' ')

             CALL       PGM(WWCONDET) PARM(&NUM &ACCT &CUSTOMER)
endpgm
````
