# .idea Context (IDE Metadata)

JetBrains IDE project metadata lives here (Copilot data migration configs, encoding settings, VCS mappings). These files do not affect runtime behavior but influence how the project opens in IDEs.

## Notes
* Safe to ignore for runtime migration; keep synchronized with actual project structure if using JetBrains products.
* No direct dependencies on source members, but updating directory structures should be reflected here to avoid IDE warnings.

## Migration Suggestions
* When moving away from JetBrains tooling, you can drop this folder. If new tooling is adopted, ensure equivalent workspace settings exist to assist future agents.
